    About

    What is this ?

    Groupie-Tracker is a student project based on an API. You will have access to information about dozens of different artists, albums, members, dates, etc.

    Who are you ?

    We are students in computer science at Ynov Nantes. Members : Guillet Evan, Fuzeau Maxime, Payraudeau Sulien, Ngono Abanda Christian.

    Technical stuff ?

    The whole project is coded in Golang, on the backend. We use a API specially created for our student project. And we work in groups with Gitea.

    Who did what ?

    Sulien : Connection api artist/date/location/relation, search system by name with error management, sorting of dates, link to different media, Spotify balise, display of data on the html pages.

    Maxime : Html/css, filtering by date of creation/number of artists, filtering by alphabetical order, display of data and responsiveness, golang server, html/server link.

    Evan : Html/Css, golang server, responsiveness, pagination by number of element 8/16/32, display of data on the html page, relation api connection, (true)README.

    Christian : (false)README, reflection on pagination, artist api connection (not implemented on the project).
